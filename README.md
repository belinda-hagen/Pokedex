## Features

- Search & browse the full National Pokédex
- Type badges with iconic Pokémon color themes
- Detailed stat breakdowns for each Pokémon
- Official artwork fetched in real-time
- Fast experience powered by Vite + React

## Tech Stack

| Technology | Purpose |
|------------|---------|
| React | UI components |
| Vite | Development environment |
| CSS | Styling |
| React Icons | Iconography |
| PokeAPI | Live Pokémon data |

## Getting Started

**Requirements:** Node.js v16+ and npm (or yarn)

```bash
git clone https://github.com/belinda-hagen/Pokedex.git
cd Pokedex
npm install
npm run dev
```

## API

This project uses [PokeAPI](https://pokeapi.co/) to fetch Pokémon names, artwork, types, stats, and more.
